 
# SkillBridge

SkillBridge is a MERN stack application focused on skill-based hiring and job readiness evaluation.

## Objective
To build a skill-centric recruitment platform where candidates are evaluated based on verified skills and project evidence rather than only resumes or academic scores.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Core Modules
- Student skill & project management
- Recruiter job posting & candidate evaluation
- Admin analytics dashboard
- Automated resume generation
- Role-based authentication

## Development Status
Project setup and architecture planning phase.
